# 로그인 화면 - 사용자 인수 테스트 (UAT)

- **화면 ID**: `login_screen`
- **화면 경로**: `/login`

## 1. 화면 개요

사용자가 이메일과 비밀번호를 사용하여 애플리케이션에 로그인하는 화면입니다. 유효성 검사, 로그인 처리, 회원가입 화면 이동 기능을 포함합니다.

## 2. 테스트 시나리오

| 테스트 ID | 테스트 설명 | 테스트 절차 | 예상 결과 | 실제 결과 | 통과/실패 |
|---|---|---|---|---|---|
| **UAT-LOGIN-001** | **정상 로그인** | 1. 올바른 이메일 입력 (e.g., `test@example.com`)<br>2. 올바른 비밀번호 입력 (e.g., `password`)<br>3. '로그인' 버튼 클릭 | 1. 로그인 성공<br>2. 대시보드 화면으로 이동 | | |
| **UAT-LOGIN-002** | **잘못된 비밀번호** | 1. 올바른 이메일 입력<br>2. 잘못된 비밀번호 입력<br>3. '로그인' 버튼 클릭 | 1. '로그인 실패' 메시지가 포함된 스낵바(알림) 표시 | | |
| **UAT-LOGIN-003** | **존재하지 않는 이메일** | 1. 존재하지 않는 이메일 입력<br>2. 비밀번호 입력<br>3. '로그인' 버튼 클릭 | 1. '로그인 실패' 메시지가 포함된 스낵바(알림) 표시 | | |
| **UAT-LOGIN-004** | **이메일 필드 유효성 검사 (입력 없음)** | 1. 이메일 필드를 비워둠<br>2. '로그인' 버튼 클릭 | 1. 이메일 필드 하단에 '이메일을 입력해주세요' 오류 메시지 표시 | | |
| **UAT-LOGIN-005** | **이메일 필드 유효성 검사 (형식 오류)** | 1. 이메일 형식에 맞지 않는 텍스트 입력 (e.g., `test`, `test@`)<br>2. '로그인' 버튼 클릭 | 1. 이메일 필드 하단에 '올바른 이메일 형식을 입력해주세요' 오류 메시지 표시 | | |
| **UAT-LOGIN-006** | **비밀번호 필드 유효성 검사 (입력 없음)** | 1. 이메일 입력<br>2. 비밀번호 필드를 비워둠<br>3. '로그인' 버튼 클릭 | 1. 비밀번호 필드 하단에 '비밀번호를 입력해주세요' 오류 메시지 표시 | | |
| **UAT-LOGIN-007** | **비밀번호 보이기/숨기기** | 1. 비밀번호 필드에 텍스트 입력<br>2. 필드 우측의 눈 모양 아이콘 클릭<br>3. 다시 아이콘 클릭 | 1. 입력된 비밀번호가 보였다가 다시 마스킹 처리됨 | | |
| **UAT-LOGIN-008** | **로그인 버튼 비활성화** | 1. '로그인' 버튼 클릭<br>2. 로그인이 진행되는 동안 '로그인' 버튼 확인 | 1. 버튼이 비활성화되고 로딩 아이콘이 표시됨 | | |
| **UAT-LOGIN-009** | **회원가입 화면 이동** | 1. '계정이 없으신가요? 회원가입하기' 텍스트 버튼 클릭 | 1. 회원가입 화면(`/register`)으로 이동 | | |
| **UAT-LOGIN-010** | **비밀번호 필드 입력 제한** | 1. 비밀번호 필드에 한글, 특수문자 등 입력 시도 | 1. 영문과 숫자만 입력됨 | | |
